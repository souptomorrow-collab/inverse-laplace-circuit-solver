# ⚡ 反拉普拉斯轉換機 (Inverse Laplace Transform · Circuit Solver)

一個**純離線、雙擊即開**的網頁工具：把 s 域函數 F(s) 反轉換回時域 f(t)，並且能**畫電路圖**自動建立 s 域模型求解（含初始條件、互感、二極體、開關、脈波源、多波形比較、**多探針疊圖**、**週期穩態 PSS**）。切換式電源（Buck／同步 Buck）可直接求穩態。

## 怎麼開始
用瀏覽器（Edge / Chrome / Firefox）打開 **`index.html`** 即可，免安裝、免連網。
> `index.html` 與 `engine.js` 必須放在**同一個資料夾**。

---

## 📐 公式反轉換
輸入 F(s) → f(t)（t ≥ 0）＋波形圖＋極點/穩定性分析。核心用**部分分式 / 留數法**。

支援：`+ − * / ^`、括號、隱含乘法（`2s`、`(s+1)(s+2)`）、`pi`、時間延遲 `exp(-2*s)` 或 `e^(-2s)`。變數用小寫 `s`。

| 輸入 | f(t) |
|---|---|
| `(s+3)/(s^2+2*s+5)` | `e^(−t)·(cos(2t) + sin(2t))` |
| `1/(s+1)^2` | `t·e^(−t)` |
| `(s^2+1)/(s+1)` | `δ'(t) − δ(t) + 2·e^(−t)` |
| `exp(-2*s)/(s+1)` | `e^(−(t−2))·u(t−2)` |

## 🔌 電路求解

### 畫電路
工具列**只放元件**（R / L / C / 電壓源 V / 電流源 I / 二極體 D / 開關 SW / 導線 / 接地），在格點上**點兩個端點**放置（接地點一下）。畫布會**填滿視窗寬度自動縮放**；放置時有**虛線預覽**跟著游標、游標下格點高亮。

### 編輯（選取/移動工具）
- **框選**：在空白處拖曳出方框可一次選多個元件（按 <kbd>Shift</kbd> 點選為加選/取消）。
- **移動**：拖元件本體移動（多選時一起移動）。
- **改尺寸 / 旋轉任意角度**：選取後拖兩端的**綠色端點**即可改變長度與方向；按住 <kbd>Alt</kbd> 可拖到**任意角度/長度**（自由角度的端點不吸附格點，需自行對齊才會相連）。<kbd>R</kbd>（或「⟳ 旋轉」鈕）為 90° 旋轉。
- **複製 / 剪下 / 貼上**：<kbd>Ctrl+C</kbd> / <kbd>Ctrl+X</kbd> / <kbd>Ctrl+V</kbd>（或工具列按鈕）；貼上的元件自動取新編號、可直接拖曳定位。
- <kbd>Del</kbd>（或「✕ 刪除選取」鈕）刪除；<kbd>Ctrl+Z</kbd> **復原**；<kbd>Esc</kbd> 隨時**回到選取/移動**。
- **探針也視為元件**：選取後可拖端點/本體移動、<kbd>R</kbd> 旋轉、<kbd>Del</kbd> 刪除（電流探針綁定在元件上）。
- 元件之間**不可重疊**（共線重疊會被擋下）。

### 量測與設定（右側面板，非元件操作都在這裡）
- **電壓探針**：按「＋電壓探針」後在畫布點兩個節點（輸出 V₁−V₂）。
- **電流探針**：用下拉選單選元件即可加入（沿元件 a→b 方向，畫布上有參考箭頭）。
- **方向性**：選取探針後按 <kbd>F</kbd>（或面板「⇄」鈕）可**反向**——電壓探針交換 +/−、電流探針反轉箭頭，數值隨之變號。
- **可同時放多個探針**（電壓、電流混用）→ 自動疊圖比較、各一色，清單可逐一移除。
- **節點初始電壓**：按「＋指定節點」點一下格點，再於面板輸入電壓（不用彈窗）。
- **互感**：選兩個電感按「建立耦合」，以 k 或 M 設定。

按「求解輸出」→ 得到轉移函數 F(s)、時域響應 f(t) 與波形；求解較久時會顯示**進度條（百分比）**。

波形圖可**互動**：**拖曳平移**、**滾輪縮放**（以游標為中心）；按「⤢ 重置縮放」或**雙擊**圖面即可恢復原本視窗。

### 進階功能
| 功能 | 說明 |
|---|---|
| **初始條件** | 電容設 v(0)、電感設 i(0)（元件參數面板）；或用「節點初始電壓」工具設定節點電壓，自動套到相連電容。 |
| **互感 M** | 選「互感 M」工具點兩個電感建立耦合，右側面板以**耦合係數 k** 或**互感 M**（可切換）設定；k 正負代表同名端方向。 |
| **二極體** | 理想二極體，用**狀態假設＋事件偵測**自動判斷導通（分段拉氏）。可設**順向壓降 V_f**（0＝理想）。 |
| **開關** | 設初始狀態（開/閉）＋切換時間 t_s，或 **PWM 週期**（T、duty）。可設**導通電阻 R_on / Rds(on)**（0＝理想）；PWM 可勾**互補**作同步整流下臂。 |
| **同步 Buck** | 上臂 PWM 開關 + **互補**下臂開關（取代飛輪二極體）即可組出同步 Buck。 |
| **週期穩態 (PSS)** | 勾「穩態 (PSS)」→ 以**打靶法**直接解切換週期的週期穩態（不模擬冗長暫態），顯示數個乾淨的穩態週期。**切換式電源（Buck 等）建議用此模式**。 |
| **脈波源** | 電壓/電流源波形選「脈波」，設週期 T_s 與工作比 duty（用平移步階響應疊加）。 |
| **多波形比較** | 單一探針時，勾選多種波形套用到第一個電壓源並**疊圖比較**。 |

內建範例：RC 低通、RLC 串聯、半波整流（二極體）、變壓器（互感）、**Buck(二極體)**、**同步 Buck**、**Boost(升壓)**。

## 演算法
1. **公式 / 輸出 F(s)** → 化為有理函數（解析器處理隱含乘法、次方、延遲）。
2. **求根**：Durand–Kerner；**重根**用 Newton 在 D⁽ᵐ⁻¹⁾ 上精修到機器精度。
3. **零極點對消**：分子分母各自分群比對、取最小重數對消、重建乾淨多項式。
4. **部分分式**：線性方程組一次解出所有留數（含重根、複數極點）。
5. **時域**：`1/(s−p)^k → t^(k−1)/(k−1)!·e^(pt)`、複數共軛 → 衰減弦波、improper → δ。
6. **電路**：s 域修正節點分析 (MNA)。初始條件用伴隨模型（電容 sC ∥ C·v₀；電感 1/sL ∥ i₀/s）；互感用電感矩陣反矩的耦合導納；二極體/開關用**分段拉氏**（每段 LTI、邊界擷取電容電壓與電感電流作為下段初始條件、時變源相位在各段重新表示）；脈波用平移步階疊加。二極體可帶 V_f（導通段為 V_f/s 電壓源），開關可帶 R_on（導通段為電導）。
7. **時間正規化**：µs 等級電路（極點 ~10⁵–10⁶）會讓求根/部分分式失準。引擎自動以 t=T₀·τ 變數變換（C→C/T₀、L→L/T₀、源頻率/時間隨之縮放），把極點拉到 O(1) 求解，再還原 v(t)=v_norm(t/T₀)（數值完全等價）。
8. **週期穩態 (PSS)**：以打靶法（Newton on F(x)=P(x)−x，P 為單一切換週期的狀態映射）直接求週期穩態 x*，避免模擬數百個切換週期的暫態。切換型開關電路為仿射映射、一步收斂；含二極體則數步收斂。

## 正確性驗證
- **演算法**先以 Python + **SymPy / 數值積分**對照（`proto*.py`）：反轉換、MNA、解析器、求根、**初始條件、互感、分段開關、二極體事件偵測、脈波**全部達機器精度。
- **JS 實作**以無頭 Edge 瀏覽器實跑驗證：反轉換 52 例 + 電路擴充 7 例 + UI 互動 10 例全數通過。
- 打開頁面後可在「📖 說明 / 自我測試」分頁即時驗證。

## 範圍與限制
- ✅ 任意階有理函數 + 時間延遲；R/L/C/電源 + 初始條件 + 互感（皆 LTI，嚴謹）。
- ✅ 切換式電源（Buck／同步 Buck）的**週期穩態**用 PSS 打靶法直接求得（精確、快速）。
- ⚠️ **拉氏法本質只適用線性非時變 (LTI)**。二極體（非線性）、開關（時變）以**理想化＋分段拉氏**處理（可加 V_f / R_on）；多二極體複雜組合可能受限。
- ⚠️ **長暫態**：切換電路的啟動暫態需模擬數百個切換週期（每段一次求解），計算較久——若只需穩態值，勾**穩態 (PSS)** 可直接求得、快很多。暫態與穩態結果一致（Buck 暫態與 PSS 皆收斂到 5V）。
- ⚠️ 少量非有理特殊轉換查表（`1/√s`、`1/√(s²+a²)` → J₀）；真正任意符號超出純瀏覽器範圍。
- ⚠️ 重根偵測對相距 < ~1% 的相異極點可能誤判為重根（數值方法固有限制）。
